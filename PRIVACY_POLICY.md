# プライバシーポリシー / Privacy Policy

**最終更新日 / Last Updated**: 2025年1月10日 / January 10, 2025

## 日本語

### 概要

Favorite Keyword Link Frog（以下「本拡張機能」）は、ユーザーのプライバシーを尊重します。本プライバシーポリシーは、本拡張機能がどのような情報を収集し、どのように使用するかを説明します。

### 情報の収集

**本拡張機能は、いかなる個人情報も収集しません。**

本拡張機能は以下のデータのみを扱います：

- **書き換えルール設定**: ユーザーが設定した正規表現パターンやリンク先URL
- これらのデータは**すべてブラウザのローカルストレージに保存**されます
- **外部サーバーへの送信は一切行いません**

### データの使用目的

保存されたデータは、以下の目的でのみ使用されます：

- ユーザーが設定した書き換えルールの保存と管理

### データの共有

**本拡張機能は、第三者とデータを共有することはありません。**

すべてのデータはユーザーのブラウザ内にのみ保存され、外部に送信されることはありません。

### データのセキュリティ

本拡張機能は、Chromeの標準的なストレージAPIを使用してデータを安全に保存します。データはユーザーのデバイス上にのみ存在し、開発者や第三者がアクセスすることはできません。

### ユーザーの権利

ユーザーは以下の権利を有します：

- いつでも本拡張機能をアンインストールできます
- アンインストールにより、保存されたすべてのデータが削除されます
- 拡張機能の設定画面から、保存されたデータを手動で削除できます

### プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて更新されることがあります。重要な変更がある場合は、本ページにて通知します。

### お問い合わせ

本プライバシーポリシーに関するご質問は、GitHubリポジトリのIssueセクションにてお問い合わせください：
https://github.com/akAredminEogre/favorite-keyword-link-frog

## English

### Overview

Favorite Keyword Link Frog (hereinafter "this extension") respects user privacy. This Privacy Policy explains what information this extension collects and how it is used.

### Information Collection

**This extension does not collect any personal information.**

This extension only handles the following data:

- **Rewrite Rule Settings**: Regular expression patterns and link URLs configured by users
- All this data is **stored in the browser's local storage**
- **No data is transmitted to external servers**

### Purpose of Data Use

Stored data is used only for the following purposes:

- Providing functionality to add links to keywords on web pages
- Saving and managing rewrite rules configured by users

### Data Sharing

**This extension does not share data with third parties.**

All data is stored only within the user's browser and is never transmitted externally.

### Data Security

This extension uses Chrome's standard storage API to securely store data. Data exists only on the user's device and cannot be accessed by developers or third parties.

### User Rights

Users have the following rights:

- Users can uninstall this extension at any time
- Uninstalling will delete all stored data
- Users can manually delete stored data from the extension's settings screen

### Changes to Privacy Policy

This Privacy Policy may be updated as necessary. If there are significant changes, they will be announced on this page.

### Contact

For questions regarding this Privacy Policy, please contact us through the Issues section of the GitHub repository:
https://github.com/akAredminEogre/favorite-keyword-link-frog
