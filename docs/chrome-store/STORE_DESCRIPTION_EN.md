# Chrome Web Store Description

## Short Description (132 characters or less)
A Chrome extension that modifies DOM elements on web pages displayed in your browser.
Freely manipulate DOM using regular expressions.

## Detailed Description
Make your browsing wishes come true - turn specific strings into links, hide elements that match certain conditions, and more.
Just like the find-and-replace function in a text editor, simply specify the before and after DOM in HTML format and save.
Streamline tedious tasks like searching for product numbers on e-commerce sites, clicking links to access them, and adding personal notes to product pages.

### Overview
Frog Frame Front is an extension that manipulates DOM displayed in your browser, enabling element rewriting and string linkification. It allows flexible DOM manipulation using regular expressions.

### Key Features
- **DOM targeting with regular expressions**: Use regular expressions to target DOM elements and enable element rewriting
- **Rule saving and management**: Save created rewrite rules and manage them per site
- **Flexible linkification**: Convert text matching arbitrary string patterns into links

### How to Use
1. Install the extension
2. Open a web page and right-click to select the area you want to modify
3. Enter replacement rules in the opened popup and save them
4. The saved rules are automatically applied, and the specified DOM is rewritten
