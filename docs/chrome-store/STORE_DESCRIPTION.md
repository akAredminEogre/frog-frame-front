# Chrome Web Store 掲載用説明文


## 短い説明（132文字以内）
ブラウザに表示中のウェブページ内のDOMを書き換えるChrome拡張機能。
正規表現で自由にDOM操作が可能です。


## 詳細な説明
特定の文字列をリンク化する、条件にあう要素を非表示にする、ネットサーフィンをするときのそんな願いを叶えます。​
メモテキストエディタの置換機能のように変更前、変更後のDOMをhtmlで指定して保存するだけ。
​ECサイトの品番をgoogleで調べ、リンクをクリックしてアクセスする、商品ページに自分用のメモをする・・・そんな面倒な作業を効率化します。


### 概要
Frog Frame Front は、ブラウザに表示したDOMを操作し、要素の書き換え、文字列のリンク化を行う拡張機能です。正規表現を利用し、自由なDOM操作が可能です。


### 主な機能
- **正規表現によるDOM指定**: 正規表現を使用してDOMを指定し、要素の書き換えが可能
- **ルールの保存と管理**: 作成した書き換えルールを保存し、サイトごとに管理できます
- **柔軟なリンク化**: 任意の文字列パターンにマッチするテキストをリンクに変換


### 使い方
1. 拡張機能をインストールします
2. Webページを開き、変更したい箇所を右クリックで選択します。
3. 開いたポップアップに、置換ルールを入力して保存します。
4. 保存したルールが自動的に適用され、指定したDOMが書き換えられます。
