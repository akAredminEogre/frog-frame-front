## 正規表現を利用したDOM置換ができること
- `置換前` : `<h1>.+</h1>`
- `置換後` : `<h2>$1</h2>`
とルールを指定すると、例えば
```html
<h1>見出し</h1>
```
が
```html
<h2>見出し</h2>
```
に置換されること
## 正規表現のUXについて
- タグ文字列はエスケープせずにユーザーが指定できるようにしたい。つまり

```html
\<h1\>.+\<\/h1\>
```
と記述しなくても

```html
<h1>.+</h1>
```
と記述すれば正規表現として認識されること
