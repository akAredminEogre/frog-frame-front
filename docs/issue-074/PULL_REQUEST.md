# ISSUE-074 PULL REQUEST

## タイトル
リライトルールパラメータ型の共通化

## 概要と理由
`App.tsx`内で直接定義されていたリライトルールのパラメータ型を、独立した型定義ファイル`RewriteRuleParams.ts`として切り出しました。

この変更により、以下のメリットが得られます：
- 型定義の再利用性向上
- コードの保守性向上
- 将来的な拡張性の向上（他のコンポーネントでも同じ型を使用可能）

## 主な変更点
1. **新規ファイル作成**
   - `src/application/types/RewriteRuleParams.ts`を作成
   - リライトルールの更新/作成時に使用するパラメータ型を定義

2. **App.tsxのリファクタリング**
   - インライン型定義を削除
   - `RewriteRuleParams`型をインポートして使用

## テスト方法
[動作確認の手順]
- `docker compose exec frontend npm run test:all` で回帰テスト通過を確認


## 補足
この変更は純粋なリファクタリングであり、機能の変更はありません。型定義を共通化することで、今後他のコンポーネントでも同じ型を使用できるようになります。
