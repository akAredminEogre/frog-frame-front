# Chrome Web Store 掲載用説明文

## 短い説明（132文字以内）
ブラウザに表示中のウェブページ内のDOMを書き換えるChrome拡張機能。正規表現で自由にDOM操作が可能です。

## 詳細な説明

### 概要
Favorite Keyword Link Frogは、ブラウザに表示したDOMを操作し、要素の書き換え、文字列のリンク化を行う拡張機能です。正規表現を利用し、自由なDOM操作が可能です。

### 主な機能
- **正規表現によるDOM指定**: 正規表現を使用してDOMを指定し、要素の書き換えが可能
- **ルールの保存と管理**: 作成した書き換えルールを保存し、サイトごとに管理できます
- **柔軟なリンク化**: 任意の文字列パターンにマッチするテキストをリンクに変換

### 使い方
1. 拡張機能をインストールします
2. 拡張機能のアイコンをクリックして設定画面を開きます
3. 正規表現パターンとリンク先URLを設定します
4. ルールを保存します
5. 対象のウェブページを開くと、自動的にリンクが適用されます

### 技術的な特徴
- React 18.3とTypeScript 5.6で開発
- Clean Architectureに基づいた設計
- 高品質なコードとテストカバレッジ

### プライバシー
この拡張機能は、ユーザーのプライバシーを尊重します。詳細は[プライバシーポリシー](プライバシーポリシーのURL)をご覧ください。
