# 概要
<!-- このチケットで解決したい課題 -->
調査により、現時点でのこの拡張機能は、
```
  private getCurrentSelection(): Selection | null {
    return window.getSelection();
  }
```
でNULLを返さないことがわかりました。(別の仕組みにより選ばれていない状態でこのメソッドを呼び出せないため)
なので、getCurrentSelectionの戻り値からNULLを外してください。

同様に、このメソッドの戻り値を受け取っている箇所も戻り値からNULL型を外し、NULLチェックも外してください。
それ以降のメソッドでも再帰的に同様の処理をしてください。
