# Chrome拡張機能の読み込み・動作確認ガイド

## 概要

生成されたzipファイル (`frog-frame-front-0.1.1-chrome.zip`) をChrome拡張機能として読み込み、動作確認を行う手順を説明します。

## 前提条件

- Google Chrome ブラウザがインストールされていること
- 開発者モードが有効になっていること

## 手順

### 1. Chrome拡張機能管理画面を開く

1. Google Chromeを開く
2. アドレスバーに `chrome://extensions/` と入力してEnterキーを押す
3. 拡張機能管理画面が表示される

### 2. 開発者モードを有効にする

1. 拡張機能管理画面の右上にある「開発者モード」トグルをONにする
2. 新しいボタンが表示される（「パッケージ化されていない拡張機能を読み込む」等）

### 3. zipファイルを展開する

**方法A: zipファイルを直接ドラッグ&ドロップ（推奨）**
1. 生成されたzipファイル `frog-frame-front-0.1.1-chrome.zip` をエクスプローラーで表示
2. zipファイルを拡張機能管理画面にドラッグ&ドロップする

**方法B: 手動で展開してから読み込み**
1. zipファイルを任意のフォルダに展開する
   - 展開先例: `C:\temp\frog-frame-front-0.1.1\`
2. 拡張機能管理画面で「パッケージ化されていない拡張機能を読み込む」をクリック
3. 展開したフォルダを選択する

### 4. 拡張機能の確認

1. 拡張機能が正常に読み込まれると、拡張機能一覧に「frog-frame-front」が表示される
2. 拡張機能のアイコンがChromeのツールバーに表示される
3. エラーが表示される場合は、manifest.jsonの内容を確認する

### 5. 動作確認

#### 基本動作確認
1. **ポップアップの表示確認**
   - ツールバーの拡張機能アイコンをクリック
   - ポップアップが正常に表示されることを確認

2. **コンテンツスクリプトの動作確認**
   - 任意のWebページを開く
   - 拡張機能のコンテンツスクリプトが正常に動作することを確認

3. **設定画面の確認**
   - 拡張機能のオプション画面が正常に表示されることを確認

#### 機能別動作確認
1. **DOM操作機能**
   - テストページでDOM要素の選択と置換が正常に動作することを確認
   - 設定したルールが正しく適用されることを確認

2. **ストレージ機能**
   - ルールの保存・読み込みが正常に動作することを確認
   - データの永続化が正しく機能することを確認

### 6. トラブルシューティング

#### よくある問題と対処法

**問題: 拡張機能が読み込めない**
- manifest.jsonの構文エラーを確認
- 必要なファイルが不足していないか確認
- Chromeの開発者ツールでコンソールエラーを確認

**問題: アイコンが表示されない**
- iconファイル（16px, 48px, 128px）が正しく配置されているか確認
- manifest.jsonのiconパスが正しいか確認

**問題: コンテンツスクリプトが動作しない**
- manifest.jsonのcontent_scriptsの設定を確認
- 対象URLのパターンが正しいか確認
- permissions設定を確認

### 7. デバッグ方法

1. **バックグラウンドページのデバッグ**
   - 拡張機能管理画面で「バックグラウンドページ」をクリック
   - 開発者ツールでエラーやログを確認

2. **コンテンツスクリプトのデバッグ**
   - 対象Webページで開発者ツール（F12）を開く
   - Consoleタブでエラーやログを確認

3. **ポップアップのデバッグ**
   - ポップアップを右クリック→「検証」
   - 開発者ツールでポップアップのHTMLとJavaScriptを確認

## 期待される動作

- ポップアップが正常に表示される
- DOM要素の選択と置換機能が動作する
- ルールの保存・編集・削除が正常に動作する
- 各種設定画面が正常に表示・動作する

## 注意事項

- 開発版の拡張機能のため、Chrome Web Storeからインストールした場合と動作が異なる可能性があります
- セキュリティ警告が表示される場合がありますが、開発版では正常な動作です
- 問題が発生した場合は、開発者ツールのコンソールログを確認してください
