のhtmlで表されるページの `商品番号：moge-1234` のテキストを選択して右クリックしたときの挙動を修正したい。

## 再現
1. 上記のHTMLを含むページを開く
2. `商品番号：moge-1234` のテキストを選択する
3. 右クリックメニューを表示する
4. `この要素を置換` を選択する

## 現在の挙動
1. 拡張機能のポップアップが開く
2. `置換前` のフォームに下記のhtmlが反映されてしまう
```html
<table class="hoge"><tbody><tr><th><span>"商品番号""："</span></th><td><span>moge-1234</span></td></tr></tbody></table>
```

## あるべき挙動
1. 拡張機能のポップアップが開く
2. `置換前` のフォームに下記のhtmlが反映されてしまう
```html
<tr><th><span>"商品番号""："</span></th><td><span>moge-1234</span></td></tr>
```
