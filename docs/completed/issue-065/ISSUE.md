# 概要
<!-- このチケットで解決したい課題 -->
RewriteRuleを編集する機能を作成する

## 画面の見た目
現在のpopup/App.tsxと同じにする

## ユースケース
1. rules.htmlで、編集したいルール行の編集をクリック
2. popup/App.tsxと同じ見た目のモーダルが表示される
3. ルールの各項目がポップアップに編集可能状態で表示される
4. 保存をクリックすると、rules.htmlのルール行が更新される。ポップアップは閉じる。編集後のルールが該当するタブがある場合には、そのタブの内容も更新される
5. キャンセルをクリックすると、ポップアップは閉じる

## 実装方針
- [ ] storybookを利用して、App.tsxのUIをatonic designで再現する
- [ ] 新規登録画面(現在のpopup/App.tsx)と編集画面は、共通部品を使うものの、別コンポーネントとする
- [ ] rules.htmlのルール行の編集をクリックしたときに、編集画面を表示する
- [ ] 編集画面で保存をクリックしたときに、rules.htmlのルール行を更新する
- [ ] 編集画面でキャンセルをクリックしたときに、ポップアップを閉じる
