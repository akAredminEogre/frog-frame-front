## ユースケース：
```html
<span class="inline-flex">
  "商品番号"
  "："
</span>
```
で表されるhtmlで、

`商品番号：` の部分を右クリックして、コンテキストメニューから `この要素を置換` を選択する。

## 期待する動作
ポップアップが開き、 `置換前：` のフォームに、
```
<span class="inline-flex">
  "商品番号"
  "："
</span>
```
が自動入力される。()

## 実際の動作
ポップアップが開き、 `置換前：` のフォームに自動入力されるのは、
```
商品番号：
```
とだけで、 `<span class="inline-flex">` の部分が抜けてしまう。
