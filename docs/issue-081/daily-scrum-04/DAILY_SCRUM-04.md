- プライバシーポリシーの作成（Chrome Web Storeの要件）

スクラム01-03でコード品質確認、マニフェストファイルの確認、README.mdの作成が完了しました。
次のステップとして、Chrome Web Storeへの登録に必要なプライバシーポリシーを作成します。

PLAN.mdの相談事項への回答により、GitHubリポジトリに掲載する方針が決定されています。
プライバシーポリシーには以下の内容を含めます:
- 拡張機能が収集する情報の種類（収集する場合）
- 収集した情報の使用目的
- 第三者との情報共有の有無
- ユーザーのプライバシー権利
- ポリシーの更新方法
- 連絡先情報
