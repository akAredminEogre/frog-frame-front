# ISSUE-060 PULL REQUEST

## タイトル
feat: ルールページ実装とE2Eテスト追加

## 概要と理由
Chrome拡張機能のオプションページにルール一覧表示機能を実装し、包括的なE2Eテストを追加しました。ユーザーが設定したルールを視覚的に確認できるようになり、保存機能との連携も検証可能になりました。

## 主な変更点
### 実装完了項目
- **オプションへの導線**: ルールページへのナビゲーション機能
- **ルール一覧表示のE2Eテスト**: 基本的なルール表示機能のテスト
- **保存後のE2Eテストでも確認**: 保存機能とルール表示機能の連携テスト

### 技術的変更点
- React を使用したルールページUIコンポーネントの実装
- WXT拡張機能のエントリーポイントとしてrulesページを追加
- 書き換えルール取得のためのUseCase実装
- 既存テストとの整合性を保った包括的なE2Eテスト構築
- コンソールエラー監視機能の統一化

### デイリースクラム実施状況
- **スクラム01**: ルールページの基本機能実装
- **スクラム02**: E2Eテスト実装（3回のレビューサイクル）
- **スクラム03**: 保存機能連携テストの完成
