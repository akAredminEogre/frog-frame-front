## 本スクラムの作業予定
Chrome Web Store審査拒否対応として、未使用のscripting権限をmanifest.jsonから削除する作業に取り組みます。

## 修正予定ファイル

- 関連するTypeScript設定ファイル（必要に応じて）

## スクラム内残タスク
- [ ] 現在のmanifest.jsonを確認し、scripting権限の使用状況を調査

- [ ] manifest.jsonからscripting権限を削除
- [ ] 削除後の動作確認とテスト実施
- [ ] 他の権限についても使用状況を簡易確認
